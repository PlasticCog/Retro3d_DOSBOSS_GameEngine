1. A simplistic 3d modeler where we can use primitive shapes, and edit them by clicking and dragging vertices. Also the ability to add sprites and scale them.

2. Animate the models by keyframes.

3. Simple texture mapping, nothing fancy.

4. Also, a scene editor where we can design a map and place the models we've created.

5. Game logic. I want to be able to control the movement, animations, creation and destruction of objects, play and stop sounds, basically a whole game programming environment that is the same as Clickteam Fusion, with the spreadsheet style of programming and controlling the objects and camera movements and sound effects/music.

6. I am aiming to be able to create games like early DOS 3d style games, like the Freescape engine and other similar style games like Seal Team and Hunter on the Amiga.

7. The ability to compile these games to executables for Windows and Linux, possibly Mac and Android as secondary goal.
